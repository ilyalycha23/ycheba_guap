Одностраничный сайт-портфолио и лендинг для веб-студии **«Сайт?Легко!»**. Курсовой проект.

---

## О проекте

Сайт представляет собой SPA (Single Page Application): весь контент подгружается в одну страницу, навигация работает без перезагрузки. Реализованы разделы: главная, портфолио, услуги, блог, контакты. Форма обратной связи отправляет сообщения в Telegram через Bot API.

### Разделы

| Раздел | Описание |
|--------|----------|
| **Главная** | О студии, статистика, навыки, преимущества, отзывы (слайдер) |
| **Портфолио** | Кейсы с описанием и технологиями (например: ООО «Атлас Барчо», ООО «Асюг», SHOP КРАСС) |
| **Услуги** | Услуги с ориентировочными сроками и ценами (интернет-магазин, многостраничный сайт, лендинг, визитка) |
| **Блог** | Список статей и полноценный просмотр поста (post1–post5) |
| **Контакты** | Форма обратной связи (отправка в Telegram), ссылки на соцсети |
| **Документы** | Политика конфиденциальности и пользовательское соглашение |

## Технологии

- **HTML5**, **CSS3**, **JavaScript** (ES5+)
- **React 18** (UMD через CDN) — только для верхнего меню и бургер-навигации
- **Babel Standalone** — компиляция JSX в браузере
- Без сборки и бэкенда: данные в `data/*.js`, контент строится в `script.js`

## Запуск

Установка зависимостей не требуется: библиотеки подключаются по CDN.

### Вариант 1: открыть файл

Откройте `index.html` в браузере (через проводник или перетаскиванием).

### Вариант 2: локальный сервер (рекомендуется)

Из корня проекта выполните:

```bash
# Node.js
npx serve .

# или Python 3
python -m http.server 8000
```

Затем откройте в браузере адрес, который выведет команда (например, `http://localhost:3000` или `http://localhost:8000`).

## Форма обратной связи и Telegram

Форма на странице **Контакты** отправляет сообщения в Telegram через [Bot API](https://core.telegram.org/bots/api).

Чтобы получать заявки в свой чат:

1. Создайте бота через [@BotFather](https://t.me/BotFather) и скопируйте токен.
2. Узнайте свой Chat ID (например, через [@userinfobot](https://t.me/userinfobot) или при первом сообщении боту).
3. В файле **`script.js`** найдите переменные `TOKEN` и `CHAT_ID` (около строк 673–674) и подставьте свои значения.

```javascript
const TOKEN = 'ВАШ_ТОКЕН_БОТА';
const CHAT_ID = 'ВАШ_CHAT_ID';
```

⚠️ **Не публикуйте токен бота в открытых репозиториях.** Для GitHub лучше использовать placeholder и хранить реальные значения в переменных окружения или секретах.

## Изображения

В папке `images/` должны лежать картинки, на которые ссылаются данные в `data/*.js`, например:

- `about-us-me.png` — фото на главной
- `sait/*.jpg` — скриншоты проектов
- `blog-*.jpg` — превью постов блога

Если каких-то файлов нет, блоки с картинками могут отображаться пустыми или с битой ссылкой.
