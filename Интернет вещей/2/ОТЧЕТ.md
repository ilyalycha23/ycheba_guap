# Лабораторная работа №2
## Создание веб-приложения на Flask

**Вариант:** Умное игровое поле для игры в баскетбол

---

### Цель работы
Создать веб-приложение на Flask с классами Thing и Sensor для IoT-системы умного баскетбольного поля.

---

### Задание
Разработать веб-приложение на Flask. Реализовать абстрактный класс Thing и класс Sensor, наследуемый от него.

---

### Ход работы

1. **Установка Flask:**
   ```
   pip install flask
   ```

2. **Создание структуры проекта:**
   - `app.py` — основной файл приложения
   - `things.py` — модуль с классами Thing и Sensor
   - `templates/index.html` — шаблон главной страницы

3. **Класс Thing** — абстрактный базовый класс с методом `connect()`.

4. **Класс Sensor** — наследует Thing, представляет датчик попаданий мяча в корзину. Метод `emulate()` возвращает случайное значение 1, 2 или 3 (очки за бросок).

5. **Запуск приложения:** `python app.py` — сервер доступен по адресу `http://127.0.0.1:5000`

---

### Выводы
1. Создано базовое веб-приложение на Flask.
2. Реализованы абстрактный класс Thing и класс Sensor для датчика попаданий.
3. Приложение готово к расширению функционала эмулятором датчика с AJAX.
