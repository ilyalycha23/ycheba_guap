# Лабораторная работа №3
## Эмулятор датчика с привязкой к AJAX-запросу

**Вариант:** Умное игровое поле для игры в баскетбол

---

### Цель работы
Реализовать эмулятор датчика попаданий с обновлением данных через AJAX-запросы.

---

### Задание
Доработать веб-приложение: датчик должен отдавать данные по GET-запросу, а интерфейс обновляться через AJAX без перезагрузки страницы.

---

### Ход работы

1. **Маршрут `/connect`** — возвращает JSON с текущим значением датчика (очки за бросок).

2. **Файл `index.js`** — функция `get_data()` выполняет AJAX GET-запрос каждую секунду (`setInterval`).

3. **Параметры AJAX:**
   - `type`: GET
   - `url`: /connect
   - `dataType`: json
   - `success`: обновляет значение в поле `#value`

4. **Шаблон `sensor_emulator.html`** — поле для отображения очков за бросок, подключает `index.js` и `main.css`.

5. **Структура проекта:**
   - `static/js/index.js` — AJAX-логика
   - `static/css/main.css` — стили
   - `templates/sensor_emulator.html` — HTML-шаблон

---

### Выводы
1. Реализован эмулятор датчика с AJAX-обновлением данных каждую секунду.
2. Данные передаются в формате JSON.
3. Интерфейс обновляется без перезагрузки страницы.
