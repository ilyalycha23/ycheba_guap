# Лабораторная работа №4
## Обработка данных датчика от внешних источников

**Вариант:** Умное игровое поле для игры в баскетбол

---

### Цель работы
Реализовать приём и обработку данных датчика от внешних источников (пользовательский ввод).

---

### Задание
Доработать метод `connect` класса Sensor так, чтобы он принимал параметры через GET-запрос (request) и обрабатывал введённые пользователем данные.

---

### Ход работы

1. **Метод `connect(request)`** — получает объект `request` от Flask, извлекает параметр `value` через `request.args.get('value', '')`.

2. **AJAX-запрос** — передаёт введённое пользователем значение в параметре `data: { "value": ... }`.

3. **Кнопка «Отправить»** — вызывает `send_data()`, которая выполняет GET-запрос с переданным значением.

4. **Ответ** — возвращает JSON с полем `power` (состояние датчика: On/Off).

5. **Интерфейс** — поле ввода очков (1, 2 или 3), кнопка отправки, поле отображения состояния.

---

### Выводы
1. Реализован приём данных от пользователя через GET-параметры.
2. Метод `connect` корректно обрабатывает объект `request`.
3. Система готова к добавлению валидации введённых данных (лаба 5).
