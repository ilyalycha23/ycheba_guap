# Лабораторная работа №8
## Отображение сохранённых данных

**Вариант:** Умное игровое поле для игры в баскетбол

---

### Цель работы
Реализовать отображение данных, сохранённых в MongoDB, на веб-странице.

---

### Задание
Добавить страницу просмотра истории очков, получаемую из MongoDB с помощью метода find().

---

### Ход работы

1. **Метод `get_all_scores()` в классе Logger:**
   - Использует `db['Scores'].find({}, {'_id': 0})` для получения всех записей
   - Поле `_id` исключено для удобства отображения
   - Возвращает список словарей с полями `timeStamp` и `Points`

2. **Маршрут `/history`:**
   - Вызывает `logger.get_all_scores()`
   - Передаёт данные в шаблон `history.html`

3. **Шаблон `history.html`:**
   - Таблица с колонками «Время» и «Очки»
   - Цикл `{% for record in records %}` для вывода всех записей
   - Ссылка на главную страницу

4. **Навигация:** на главной странице добавлена ссылка «Просмотр истории очков».

---

### Выводы
1. Реализовано отображение данных из MongoDB.
2. Метод find() корректно извлекает записи из коллекции.
3. Система готова к визуализации данных с помощью Chart.js (лаба 9).
