# Лабораторная работа №9
## Визуализация сохранённых данных с Chart.js

**Вариант:** Умное игровое поле для игры в баскетбол

---

### Цель работы
Реализовать визуализацию данных датчика с помощью библиотеки Chart.js.

---

### Задание
Создать страницу с графиком, отображающим историю очков за броски. Использовать Chart.js для построения графика.

---

### Ход работы

1. **Подключение Chart.js:**
   ```html
   <script src="https://unpkg.com/chart.js@3.5.0/dist/chart.min.js"></script>
   ```

2. **API-маршрут `/api/chart_data`:**
   - Возвращает JSON с полями `labels` (время) и `data` (очки)
   - Используется для загрузки данных в график

3. **Страница `/chart`:**
   - Элемент `<canvas id="myChart">` для отрисовки графика
   - Fetch-запрос к `/api/chart_data` для получения данных
   - Инициализация `new Chart()` с типом `line`

4. **Параметры графика:**
   - `type: 'line'` — линейный график
   - `labels` — метки времени (timeStamp)
   - `datasets[0].data` — значения очков
   - `tension: 0.3` — сглаживание линий
   - Шкала Y: от 0 до 4 (очки 1, 2, 3)

5. **Навигация:** добавлена ссылка «График (Chart.js)» на главной странице.

---

### Выводы
1. Реализована визуализация данных с помощью Chart.js.
2. График отображает историю очков за броски во времени.
3. Система «Умное игровое поле» полностью реализована с полным циклом: датчик → валидация → актуатор → логгер → отображение → визуализация.
