# Лабораторные работы — Интернет вещей

**Вариант:** Умное игровое поле для игры в баскетбол

## Структура

| Папка | Описание |
|-------|----------|
| 1 | Концептуальная модель IoT (UML, отчёт) |
| 2 | Flask-приложение, классы Thing и Sensor |
| 3 | Эмулятор датчика с AJAX |
| 4 | Обработка данных от внешних источников |
| 5 | Валидация данных (try-except) |
| 6 | Интеллектуальный актуатор (табло счёта) |
| 7 | Логгер с MongoDB |
| 8 | Отображение сохранённых данных |
| 9 | Визуализация с Chart.js |

## Запуск

### Лабы 2–6
```bash
pip install flask
cd 2   # или 3, 4, 5, 6
python app.py
```
Открыть: http://127.0.0.1:5000

### Лабы 7–9 (нужен MongoDB)
```bash
pip install flask pymongo
# Запустить MongoDB (localhost:27017)
cd 7   # или 8, 9
python app.py
```

### requirements.txt
```
flask>=2.0.0
pymongo>=4.0.0
```
